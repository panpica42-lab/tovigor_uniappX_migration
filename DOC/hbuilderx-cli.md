# HBuilderX CLI

本机 HBuilderX 安装路径：

```text
C:\wzl\HBuilderX\HBuilderX.exe
```

本机 CLI 路径：

```text
C:\wzl\HBuilderX\cli.exe
```

常用命令：

```powershell
& 'C:\wzl\HBuilderX\cli.exe' project open --path 'C:\wzl\HX-proj\tovigor_v1\uniapp-X_transfering\tovigor_yei_unIappX'
& 'C:\wzl\HBuilderX\cli.exe' launch app-android --project 'C:\wzl\HX-proj\tovigor_v1\uniapp-X_transfering\tovigor_yei_unIappX' --compile true
& 'C:\wzl\HBuilderX\cli.exe' launch app-android --project 'C:\wzl\HX-proj\tovigor_v1\uniapp-X_transfering\tovigor_yei_unIappX' --compile true --cleanCache true
& 'C:\wzl\HBuilderX\cli.exe' lsp lint --project 'C:\wzl\HX-proj\tovigor_v1\uniapp-X_transfering\tovigor_yei_unIappX' --file 'C:\wzl\HX-proj\tovigor_v1\uniapp-X_transfering\tovigor_yei_unIappX\pages\idle\idle.uvue' --platform app-android
```
